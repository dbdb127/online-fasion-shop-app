# Shopping-App-Project
KAIST MADCAMP Project-2

## Development Environment
Client
- Android Studio
- Java

Server
- JavaScript
- Node.js
- Express

Database
- Mysql


## Dependencies Used
Kakao SDK
Google SDK
Retrofit
Glide

## Usage
Download or clone the project
Compile on Android studio or use apk
